# Shared UI Component Library

Welcome to the Shared UI component library! This library provides a comprehensive set of reusable components built with React, TypeScript, and Tailwind CSS, following the Atomic Design methodology.

## ğŸ¯ Philosophy

Our component library is built on these core principles:

- **Atomic Design**: Components are organized into atoms, molecules, and organisms
- **Accessibility First**: All components follow WCAG guidelines
- **Type Safety**: Full TypeScript support with comprehensive types
- **Customizable**: Easy to extend and customize with Tailwind classes
- **Performance**: Optimized for minimal bundle size and fast rendering

## ğŸ—ï¸ Structure

### Atoms
Basic building blocks that can't be broken down further:
- Button, Input, Label, Badge
- Checkbox, Switch, Separator
- Alert, Loading, Textarea, Tabs

### Molecules
Combinations of atoms that form functional units:
- Card (with Header, Content, Footer)
- Select (with dropdown functionality)
- Form (with validation support)
- DropdownMenu (contextual menus)

### Organisms
Complex components that combine molecules and atoms:
- AdminSidebar (configurable navigation)
- ThemeSelector (theme switching)
- Dialog (modal windows)
- Table (data display)
